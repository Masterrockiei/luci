## Variables

* `API_ID` - Your API_ID
* `API_HASH` - Your API_ID
* `BOT_TOKEN` - Your Bot Token
* `ADMINS` - Your Telegram ID
* `GROUP_ID` - To use the bot personally

<h2> Features 🔥 </h2>

* Support Fast Song Download 
* Supports Music Find Feature
